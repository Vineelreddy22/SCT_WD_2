 # Stopwatch Pro

## SkillCraft Technology Internship - Task 2

A modern and interactive stopwatch web application built using HTML, CSS, and JavaScript.

## Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Recording
- Fastest Lap Tracking
- Slowest Lap Tracking
- Average Lap Statistics
- Responsive Design
- Modern Dark UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
